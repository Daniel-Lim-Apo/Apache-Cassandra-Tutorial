# Apache Cassandra Tutorial By Daniel Lim-Apo

Apache Cassandra is a highly scalable, high-performance distributed database designed to handle large amounts of data across many commodity servers without any single point of failure. Below are various resources to help you understand and work with Apache Cassandra, from basic concepts to advanced configuration.

- [Apache Cassandra Official Website](https://cassandra.apache.org/_/index.html)
- [Cassandra Basics](https://cassandra.apache.org/_/cassandra-basics.html)

- [Cassandra Documentation (Latest)](https://cassandra.apache.org/doc/latest/)
